The plot above was a real-time data taken from an ultra-sonic sensor to measure the sump water level in my basement.

Up-to-date implementations:
  • Measure water depth using an ultra-sonic sensor and Arduino UNO microcontroller.
  • Write the data into a file in the SD card memory of the ethernet card.
  • Upload the file using FTP to my personal frontier.com website. I guess I am too cheap to pay for a subscription to a web hosting with PHP. Having PHP website would've been easier to implement.
  • Use Java-Script to plot the data (plot above).
  • Send the data to another Arduino microcontroller wirelessly via Xbee. The 2nd microcontroller will be placed upstairs.
  • Both controllers, in the sump and upstairs, will be powered by batteries just in case of power outage.
